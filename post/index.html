Projects

Here are a few data analytics projects that I have worked on, feel free to check these out !

Property Analysis

As part of my internship, I developed a Property Analysis Project that evaluates key factors influencing real estate decisions across cities, states, and suburbs. The project features comprehensive tables displaying house values, rental prices, school quality, crime rates, and transportation accessibility. By aggregating and analyzing this data, the tool provides valuable insights for homebuyers, investors, and urban planners. I designed an intuitive interface to present complex datasets clearly, enabling users to make informed decisions.

Bechdel Test

As a personal passion project, I created a Bechdel Test tool to assess gender representation in films based on the Bechdel Test criteria. It evaluates whether movies include at least two named female characters who discuss something other than a man. The project features data visualizations showcasing trends in gender portrayal across genres and time periods. With an intuitive interface, it enables users to explore insights into cinematic gender dynamics.

IPL Dashboard

Driven by my husband's passion for cricket, I created an IPL Dashboard to showcase key tournament statistics and insights. The project features detailed visualizations of batting and bowling stats, orange and purple cap holders, and tournament 4s/6s, among other metrics. Built with a focus on user-friendly design, it allows fans to explore team and player performance interactively. I leveraged data analysis and visualization tools to transform raw cricket data into engaging insights. This personal project strengthened my skills in data processing, dashboard development, and sports analytics.

Call Center Dashboard

Designed and developed a comprehensive Excel-based dashboard to analyze key call centre performance metrics. The dashboard features a calls trend line graph to monitor activity over time, a weekday trend bar chart to highlight peak operational days, and a dynamic chart visualizing calls and revenue by representative. Additional insights include calls segmented by customer demographics, a rating distribution histogram to evaluate customer satisfaction, and a detailed customer revenue analysis table enhanced with conditional formatting and data bars. This project showcases strong proficiency in Excel for data analysis, visualization, and interactive reporting.
